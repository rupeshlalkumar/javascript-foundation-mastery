# 1. Project — Calculator

## 1.1 Project Goal

Create a simple calculator using JavaScript variables and operators.

This project will help you understand:

```text
Variables
+
Numbers
+
Operators
+
console.log()
+
Basic Logic
````

---

# 2. Features

Your calculator should perform:

1. Addition
2. Subtraction

---

# 3. Step 1 — Create Two Numbers

Create two variables:

```javascript
const number1 = 20;
const number2 = 10;
```

---

# 4. Step 2 — Addition

Create an addition variable:

```javascript
const addition = number1 + number2;
```

Display the result:

```javascript
console.log("Addition:", addition);
```

Expected Output:

```text
Addition: 30
```

---

# 5. Step 3 — Subtraction

Create a subtraction variable:

```javascript
const subtraction = number1 - number2;
```

Display the result:

```javascript
console.log("Subtraction:", subtraction);
```

Expected Output:

```text
Subtraction: 10
```

---

# 6. Final Project Code

```javascript
const number1 = 20;
const number2 = 10;

const addition = number1 + number2;
const subtraction = number1 - number2;

console.log("Calculator Results");
console.log("------------------");

console.log("First Number:", number1);
console.log("Second Number:", number2);

console.log("Addition:", addition);
console.log("Subtraction:", subtraction);
```

---

# 7. Expected Output

```text
Calculator Results
------------------

First Number: 20
Second Number: 10

Addition: 30
Subtraction: 10
```
