# 🚀 Mini Project: Calculator (Part 2)

## Objective

Build a simple calculator that performs:

- Multiplication
- Division

This project helps you understand:

- Variables
- Number Data Type
- Arithmetic Operators
- console.log()

---

# Project Structure

```text
mini-project.js
```

---

# Step 1: Create Variables

```javascript
const number1 = 20;
const number2 = 5;
```

---

# Step 2: Multiplication

```javascript
const multiplication = number1 * number2;

console.log("Multiplication:", multiplication);
```

Output:

```text
Multiplication: 100
```

---

# Step 3: Division

```javascript
const division = number1 / number2;

console.log("Division:", division);
```

Output:

```text
Division: 4
```

---

# Complete Project

```javascript
// ==============================
// Calculator
// Multiplication & Division
// ==============================

const number1 = 20;
const number2 = 5;

// Multiplication
const multiplication = number1 * number2;

// Division
const division = number1 / number2;

// Display Results
console.log("Calculator");
console.log("--------------------");

console.log("First Number :", number1);
console.log("Second Number:", number2);

console.log("Multiplication:", multiplication);
console.log("Division:", division);
```

---
